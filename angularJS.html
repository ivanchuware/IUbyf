Empresas como Google (por ejp. Gmail y Google Docs), Telegram, YouTube o Paypal utilizan este framework de JavaScript para frontend.

AngularJs es un framework de JavaScript de código abierto, mantenido por Google, que se utiliza para crear y mantener aplicaciones web de una sola página. Su objetivo es aumentar las aplicaciones basadas en navegador de Modelo Vista Controlador (MVC), en un esfuerzo para hacer que el desarrollo y las pruebas sean más fáciles.

Los valores de las variables de JavaScript se pueden configurar manualmente, o recuperados de los recursos JSON estáticos o dinámicos.

Características:

  1. MVC (Modelo-Vista-Controlador): Es un patrón de diseño utilizado en todas las web apps modernas. Se basa en dividir la capa de lógica empresarial, la capa de datos y la capa de presentación en secciones separadas. La división en diferentes secciones se realiza para que cada una pueda ser manejada con mayor facilidad.
  2. Enlace de modelo de datos: No es necesario escribir código especial para enlazar datos a los controles HTML. Esto puede hacerlo Angular simplemente agregando algunos fragmentos de código.
  3. Escribir menos código: Angular necesita menos cantidad de código para escribir la manipulación DOM.
  4. Pruebas unitarias listas: Google ademas de Angular desarrolló un marco de pruebas llamado “karma” que ayuda en el diseño de pruebas unitarias para apps AngularJS.
  5. Al ser marco de código abierto, se espera que la cantidad de errores o problemas sea minima.
  6. Encuadernación bidireccional: AngularJs mantiene sincronizada la capa de datos y presentación. Ahora no necesita escribir código JavaScript adicional para matener los datos en su código HTML y sus datos más adelante sincronizados. AngularJs hará esto automaticamente por usted. Solo necesita especificar que control esa vinculado a que parte de su modelo.
  7. Enrutamiento: Se puede encargar del enrutamiento, lo que significa moverse de una vista a otra. Esta es la clave fundamental de las aplicaciones de una sola pagina; en la que puede moverse a diferentes funcionalidades en su aplicación web basadas en la interacción del usuario, pero aún asi permanecer en la misma página.
  8. Admite pruebas, pruebas unitarias y pruebas de integración
  9. Se extiende HTML proporcionado de sus propios elementos llamados directivas. En un nivel alto, las directivas son marcadores de una elemento DOM (como un atributo, nombre de un alemento y comentario o clase CSS) que le dicen al compilador HTML de AngularJS que adjunte un comportamiento especificado a ese elemento DOM. Estas directivas ayudan a ampliar la funcionalidad de los elementos HTML existentes para dar más potencia a su web app.

Es excelente para aplicaciones medianas y grandes

AngularJS es una versión antigua del actual Angular (también llamado Angular 2+, con AngularJS siendo la versión 1) que ya solo recibe actualizaciones para soporte básico, con el equipo de Angular recomendando actualizar a Angular 2+.

La curva de aprendizaje de Angular (no AngularJS) es desafiadora debido a que usa TypeScript en lugar de JavaScript.
